# Algo Trading System — Claude API Backend

An algorithmic trading system using Claude API for:
- Real-time market signal analysis (Indian + US equities)
- ATH breakout pattern detection with volume confirmation
- Entry/exit decision logic based on momentum filters
- Crypto prediction market timing (Polymarket BTC markets)

## Stack
- Python
- Claude API (claude-sonnet-4-6)
- Cursor IDE
